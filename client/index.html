Client

Java REST client

Provides a simple Java client to communicate with Sonatype OSS Index.

Dependency

To use from Apache Maven include dependency:

<dependencyManagement>
  <dependencies>
    <dependency>
      <groupId>org.sonatype.ossindex</groupId>
      <artifactId>ossindex-service-client</artifactId>
      <version>1-SNAPSHOT</version>
    </dependency>
  </dependencies>
</dependencyManagement>
<dependency>
  <groupId>org.sonatype.ossindex</groupId>
  <artifactId>ossindex-service-client</artifactId>
  <version>1-SNAPSHOT</version>
</dependency>

Transports

A few options for different transports are provided.

HttpUrlConnectionTransport

The HttpUrlConnectionTransport makes use of standard JRE HttpUrlConnection.

HttpClientTransport

The HttpClientTransport makes use of Apache HttpClient.

Marshalling

The default marshalling library used is Google Gson.
